# 💱 Divisas App - Cotización en Tiempo Real

Aplicación web moderna para consultar cotizaciones de monedas internacionales en pesos argentinos (ARS), con soporte para múltiples monedas, interfaz responsive y actualización automática en tiempo real.

---

## 🚀 Demo

👉 https://TU-USUARIO.github.io/TU-REPO  
👉 API Backend: https://server-divisas.onrender.com

---

## 🧠 Características

- 🌍 Soporte para +150 monedas internacionales
- 🇦🇷 Conversión automática a pesos argentinos (ARS)
- 🏳️ Visualización de banderas por país
- ➕ Selección múltiple de monedas
- ❌ Eliminación dinámica de monedas seleccionadas
- 🔄 Actualización automática cada 30 segundos
- ⚡ Fetch concurrente de datos (Promise.all)
- 📱 Diseño 100% responsive
- 🎨 UI moderna estilo fintech (glassmorphism + gradients)
- 🧠 Manejo de errores robusto

---

## 🏗️ Arquitectura
